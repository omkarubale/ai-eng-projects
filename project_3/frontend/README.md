# React + Vite

This template provides a minimal [React](https://react.dev/) frontend  React working in [Vite](https://vite.dev)

## Setup
### Install Node and NPM

#### MacOS

```shell
brew install node
```

#### Windows

- See: [Download Node.js](https://nodejs.org/en/download)

## Build Project

### Install libraries

```shell
npm install
```

### Run frontend

```shell
npm run preview
```
